1. 最先端テクノロジー (Vibe Coding)

プログラミングの文法を丸暗記させる旧来の教育は行いません。当スタジオでは、「MCP (Model Context Protocol)」と「ClaudeCode」等の最新AIツールをバックエンドに接続し、子どもたちが「要望をAIに伝えるだけ」でシステムが組み上がる『Vibe Coding』の圧倒的ワークフローを提供します。

想像したものを形にするアプローチを最速で体験させ、AIの効果的・倫理的な使い方も同時に学ばせます。

【実現可能な学習・開発領域】

Game Roblox / Minecraft

MCPを通じゲーム内APIやMod環境をClaudeCodeに解析させ、「こんなルールにしたい」とAIに話すだけでLuaやJavaコードを生成・配置。遊びながらアルゴリズムの基礎を学びます。

3D Unity

本格的な3Dエンジンも、C#スクリプトや物理演算処理の雛形をAIがサポート。複雑なデプロイ処理もClaudeコマンド経由で自動化し、クリエイティビティに集中させます。

Dev HP・アプリ開発

Vercel等の環境と連携するMCPを用意し、要件定義からデプロイまで一気通貫。自営業の親のお店のHPを作るなど、実社会の課題解決をAIと共に行います。

Data DB・API連携

Supabase等のデータベースと各種APIをMCPで接続。データ構造の設計やバックエンド処理など、大人顔負けのシステム構築の裏側をAI経由で体感・習得させます。

Media グラフィック・3DCG・動画編集

画像生成AIによる素材作成や動画の自動カット編集を活用。さらに、画面内のデータを3Dプリンタやレーザーカッターで「物理的な形」にするFab体験も提供します。

Marke SNS・ブログ・eスポーツ指導

AIを活用したブログ集客やSNSインサイト分析を実践。また、eスポーツのプレイ動画をAIに分析させ、チームの戦略立てやコーチングなど論理的思考力を養います。

Startup 起業・自己実現

ただ作るだけでなく、それをどう社会に役立てるか。自らが見つけた「好き」でプロダクトを作り、それを堂々とプレゼンテーションして、自信を持って世界へ発信するアントレプレナー精神の終着点です。



2. スポーツ・息抜き (脳科学的アプローチ)

PC作業による座りっぱなしや依存を防ぎ、「脳をリフレッシュし、地頭を良くする」ための革新的なスポーツプログラムを取り入れます。プロアスリートも利用する最先端機器を導入します。

Cognitive ビジョントレーニングボード

ランダムに光るボタンを素早くタッチして消していくことで、周辺視野、動体視力、瞬間的な判断力を鍛え、脳のワーキングメモリを拡張させます。

Physical ARサイバー・ボルダリング

プロジェクションマッピングで映し出された光や数字を追いかけて登る最新型スポーツ。状況判断と瞬発的な全身運動を同時に行い脳を極限まで活性化させます。

Team サイバー球技アリーナ

光るボールと床を使った近未来のチームスポーツ。仲間と協調して行うチームビルディング、持続的な注意力と集中力、微細運動能力を大きく向上させます。



3. ビジネス・マネタイズ実践

学んだスキルは「おままごと」で終わらせず、シームレスに「マネタイズ（収益化）」まで繋げて社会との繋がりを実感させます。教室という安全な環境内で、起業家精神（アントレプレナーシップ）の圧倒的実践を行います。

Retail サイバー駄菓子屋 (生徒経営)

教室内にある駄菓子屋を生徒たちが仕入れ〜販売まで独自に運営。自作したタブレットPOSレジや在庫システムを稼働させ、現代版レモネードスタンドとして商売の基本を学びます。

Media YouTube・動画配信スタジオ

プロフェッショナルなマイクやグリーンバック環境を開放。自分たちの作ったゲームの実況や、商品の宣伝動画の企画・撮影・編集を行い、ファンマーケティングのリアルを実践します。

D2C 商品開発・システム受託

3Dプリンタで作ったオリジナルガジェット等の物販や、LINEミニアプリなどの業務改善システムを地域店舗などへ実際に提供・販売してみるプロセスを指導します。
