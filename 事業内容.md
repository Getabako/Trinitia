# Tech & Active Studio AKITA（仮称）事業内容

## 基本情報

- **名称案**: Tech & Active Studio AKITA（仮称）
- **コンセプト**: 「創る」と「動く」で未来を切り拓く、次世代型クリエイターズスタジオ
- **ターゲット**: 幼児〜中高生
- **所在地**: 秋田県（詳細未定）

---

## 3つのコアバリュー（提供価値）

### 1. 自然言語で形にする「最新テクノロジー教育」

単なるプログラミング学習にとどまらず、最先端の環境で実践的な開発とモノづくりを体験。

- **バイブコーディング**: 環境構築済みの Claude Code 等を活用し、自然言語による対話のみでシステムやゲームを高速開発
- **デジタルファブリケーション**: 3Dプリンターやレーザーカッターを用いた、画面の枠を超えたリアルなプロダクト制作
- **デジタルクリエイティブ**: 動画撮影スタジオ、eスポーツ環境を完備し、発信力やプレイスキルを高度なレベルで磨く

### 2. 心身を解放する「スポーツ・息抜き」

PCや機材に向き合うだけでなく、運動プログラムを取り入れることでシナジーを生み出す。

- スポーツを挟むことで脳をリフレッシュさせ、クリエイティブな発想や集中力を促進
- 最新のスポーツ科学に基づいた総合的な運動能力の開発
- チームワークを通じたコミュニケーション能力の育成

### 3. その場で価値を生み出す「マネタイズ実践」

学んだスキルを「稼ぐ」経験に直結させ、起業家精神（アントレプレナーシップ）を養う。

- 開発したアプリのリリースや、3Dプリンターで作成したプロダクトのオンライン販売
- YouTube等での動画配信や、eスポーツのスキルを活かした収益化の仕組みづくり
- 自分たちのアイデアが社会でどう評価されるかを、リアルタイムで体験するビジネス実践

## 保護者の悩みを解決

| 保護者の悩み | Tech & Active Studio の回答 |
|---|---|
| ゲームばかりしてる… | スキルを活かして収益化の勉強ができる |
| 健康のために最低限の運動もさせたい… | 体を動かす時間もある |
| 将来どんな役に立つのか… | 実際に「作って・売る」体験で起業センスが育つ |
| 学校に行き渋っている… | 将来的には出席扱いになる居場所を目指す |

**最大の魅力**: これからの未来を支えていく人材に必須の能力が身につく

**従来との差別化**:
- プログラミング教室やスポーツ教室の枠に収まらない、多面的な才能を引き出す場
- 長時間の集中力を維持する仕掛け（テック→スポーツ→テックのサイクル）

## ロードマップ

### フェーズ0: 教室開設・基盤構築（現在）

- 事業計画の策定
- 物件・設備の選定
- ホームページ制作
- 集客・認知活動

### フェーズ1: 次世代型・学童保育への展開

共働き世帯の課題解決に向け、単なる預かりではなく「スキルが身につく学童」として機能。

- テクノロジーとスポーツの両軸があるため、長時間の滞在でも飽きず、心身の健全な発達を促す環境を提供
- 幼児期から通う子どもたちが、小学生になってもシームレスに継続して学べるコミュニティの形成

### フェーズ2: 学校の「出席扱い」認定拠点の確立

心理的な安全性が高く保たれた環境を作り、学校に行きづらい子どもたちにとっての新しい学びの場を確立。

- 文部科学省のガイドラインに基づき、ITを活用した学習履歴の可視化と、在籍校の校長との連携フローの構築
- 一人ひとりの特性に合わせた独自の教育アプローチで、自分のペースでテクノロジーと運動に取り組み、社会と繋がる自信を育むサポート

### フェーズ3: 教育 × M&A モデルの構築（将来構想）

全国一の人口減少・少子高齢化という秋田県の特性を逆手に取った展開。

- 廃業危機にある老舗店舗とのコラボレーション
- 事業承継を通じた教育プログラムの拡張（マネー教育、職人の技の継承等）
- **教育 × M&A** という新しいモデルの構築
- 子どもたちが実際のビジネスに触れ、地域経済の担い手として成長する仕組み
